Tips de aplicación


  • Las fórmulas con texturas acuosas se aplican antes que las fórmulas con texturas oleosas. En caso de usar varios sérum con texturas acuosas, primero aplicar el que trata el principal problema. Por ejemplo si te preocupan más las manchas, se usaría Alpha Arbutin antes Buffet. Si necesitas controlar los granitos, Niaciamide antes que Acid Hyaluronic.

  • Puedes combinar hasta 3 sérums en cada rutina ( no debes sobrecargar la piel ) y sólo uno de ellos con un activo potente (que puede causar irritación en la piel). No usar retinol y el sérum de Vitamina C al 30% al mismo tiempo. O más de 1 ácido.

  • Se recomienda usar los activos fuertes en noches alternas.

  • No hay que usar juntos el sérum Niacinamide 10% + Zinc 1% y los diferentes sérums de vitamina C a base de ácido ascórbico (Vitamin C Suspension 30% in Silicone, Vitamin C Suspension 23%+ HA Spheres 2%, 100% L-Ascorbic Acid Powder y Ethylated Ascorbic Acid 15% Solution).

  • No usar juntos los sérums a base de ácido ascórbico (Vitamin C Suspension 30% in Silicone, Vitamin C Suspension 23%+ HA Spheres 2%, 100% L-Ascorbic Acid Powder y Ethylated Ascorbic Acid 15% Solution) con el sérum EUK.
